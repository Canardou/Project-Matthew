Simulink Support Package for AR.Drone v2.0 GYROSCOPE


This section explains how you may read the different gyroscope datas mesured by the drone.

Port map

The gyroscope block makes it possible read the three different angles of rotation (roll, pitch and yaw) mesured by the drone. Each output port gives a specific angle of rotation in degres :

  • output port 0 : roll
  • output port 1 : pitch
  • output port 2 : yaw
