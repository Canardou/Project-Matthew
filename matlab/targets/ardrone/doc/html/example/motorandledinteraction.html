Simulink Support Package for AR.Drone v2.0 Motor and LED Interaction


This example shows how the LEDs and the motor can respond to the altitude data mesured by the drone.

Model

Subsystem

Description

This model mesures the height in centimeters. The mesured data is then used to apply a thrust proportional to the drone's altitude.

Above 100 centimeters, full thrust is applied.

The LEDs are given different colours depending of the drone's altitude :

  • 0-25cm : OFF
  • 25-50cm : RED
  • 50-75cm : GREEN
  • 75-100cm : ORANGE
  • after 100cm : OFF
