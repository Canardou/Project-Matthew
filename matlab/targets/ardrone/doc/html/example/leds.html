Simulink Support Package for AR.Drone v2.0 LEDS


This section explains how you may control the different LEDs individually on the drone.

Port map

The LED block makes it possible control all four LEDs on the drone. Each input port controls a specific LED :

  • input port 0 : LED 0 (left front)
  • input port 1 : LED 1 (right front)
  • input port 2 : LED 2 (right back)
  • input port 3 : LED 3 (left back)
The following image shows where the LEDs are located on the AR.Drone v2.0.

Parameters

The LEDs are controlled by sending one of the following values on each port :

  • 0 : OFF
  • 1 : RED
  • 2 : GREEN
  • 3 : ORANGE (red and green simultaneously)
