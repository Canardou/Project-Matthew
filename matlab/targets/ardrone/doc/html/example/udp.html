Simulink Support Package for AR.Drone v2.0 UDP


This section explains how to send and receive data over a wireless network by udp.

Port map

The UDP block makes it possible send and receive data over a wireless network by udp. The mapping is the following :

  • input port 0 : data to be received
  • output port 0 : data to be sent

The data is sent and received on port 9000.
The sent data is broadcasted to IP address 192.168.1.255.
